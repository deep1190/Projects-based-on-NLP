# Scraping Instagram with Python

This project demonstrates how to scrape Instagram data using the `instaloader` Python library.

## Features
- Extract profile information: username, ID, posts, followers, followees, bio
- Scrape followers and followees list
- Download posts from any public profile
- Optional: Login to Instagram for private methods

## Python Packages Used
- `instaloader` (install via `pip install instaloader`)
