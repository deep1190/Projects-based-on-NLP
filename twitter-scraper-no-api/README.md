# Scraping Twitter with Python without API

This project demonstrates how to scrape Twitter data without using the official API, using the `twint` library in Python.

## Features
- Scrape followings of a list of Twitter users
- Analyze relationships among users
- Find most followed accounts within a user group
- Visualize relationships in a pandas DataFrame

## Python Packages Used
- `twint` (install via `pip install twint`)
- `pandas`
- `collections`
