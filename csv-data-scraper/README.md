# Web Scraping to Create CSV

This project demonstrates how to scrape product data from Flipkart using **BeautifulSoup** and **urllib** and store the data in a CSV file.

## Features
- Scrape product name, price, and ratings
- Save the scraped data to `products.csv`
- Example used: Samsung mobiles on Flipkart

## Python Packages Used
- `BeautifulSoup` (install via `pip install bs4`)
- `urllib` (standard library)
- `csv` (standard library)
